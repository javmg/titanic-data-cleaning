2. Integración y selección de los datos de interés a analizar.¶

A continuacin presento los atributos:

  • survival: booleano (0 - no y 1 - s).
  • pclass: string con clase del billete (1 - primera, 2 - segunda y 3 - tercera)
  • sexo: string (male y female)
  • edad: decimal representando la edad en aos.
  • sibsp: numrico con el nmero de hermanos/ hermanas/ cnyuges en el barco.
  • parch: numrico con el nmero de padres/ hijos en el barco.
  • ticket: numrico con el id de billete.
  • fare: numrico con el precio del billete.
  • cabin: numrico con el id de cabina.
  • embarked: string con el puerto de embarque (C, Q, S)
